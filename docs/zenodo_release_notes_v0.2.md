# Release notes: v0.2-neurocomputing

This release archives the reproducibility package associated with the manuscript:

**Validator-Backed Auditable Robust Aggregation for Trustworthy Federated Learning**

It includes the paper-specific implementation, benchmark configurations, consolidated result tables, paper figures, supplementary tables, validator-audit outputs, and audit-case excerpts used to support the reported analyses.

## Scope

- Target manuscript version: Neurocomputing submission package.
- Main artifact type: research software and reproducibility package.
- Public benchmark datasets: MNIST, Fashion-MNIST, and CIFAR-10 are loaded through standard dataset loaders and are not redistributed.
- Full raw run directories are not included in the compact package; released manifests and scripts can regenerate them.
