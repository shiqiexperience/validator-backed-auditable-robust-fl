# Release notes: v0.3-neurocomputing

This release archives the reproducibility package associated with the manuscript:

**Validator-Backed Auditable Robust Aggregation for Trustworthy Federated Learning**

Compared with the previous package metadata, this release updates the repository naming and citation metadata from the earlier hash-chain-oriented project name to:

`validator-backed-auditable-robust-fl`

The renamed repository better reflects the manuscript's current contribution: validator-backed auditability and robust aggregation for trustworthy federated learning.

Archived release DOI: https://doi.org/10.5281/zenodo.20790584
